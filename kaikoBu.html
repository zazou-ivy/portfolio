Kaîko-bu

Présentation du concept

Kaîko-bu est un réseau social développé en PHP réalisé au cours de la formation Passerelle numérique permettant aux utilisateurs de publier leurs opening d'anime favoris.

Objectifs

Le réseau social doit :

  • posséder un système d'inscription au site web
  • posséder un système de login-logout pour connecter/déconnecter l'utilisateur
  • une fois connecté, l'utilisateur devra être en mesure de publier un post
  • contenir des publications avec plusieurs informations (titre, genre, auteur, etc.)

Technologies

Voir le site Source Github

Structure du projet

Kaiko-bu a été développé entièrement avec le language de programmation PHP en POO (programmation d'objet orienté objet) via une structure MVC (Model-Vue-Controller).
Concernant les données, elles ont été générées grâce à l'interface ORM via un fichier JSON pour pouvoir afficher les données avec les vues à l'aide de Bootstrap.

Fonctionnalités

Accueil

La page d'accueil possède tous les openings postés par les utilisateurs. Elle est visible par tous les visiteurs du site, y compris ceux qui ne sont pas inscrits.

Inscription

L'inscription permet aux utilisateurs de publier leur meilleurs opening d'anime. Cette inscription recquiert un nom d'utilisateur d'au minimum 4 caractères et un mot de passe de 8 caractères.

Authentification

Pour se connecter, il faut avoir préalablement réalisé l'étape d'inscription au site. L'identifiant et mot de passe inséré devra ainsi correspondre à ceux crées lors de l'étape d'inscription.

Publication

Une fois connecté, l'utilisateur peut publier un opening. Il doit cependant renseigner tous les champs indiqués qui sont :

  • le nom de l'anime
  • le titre de la chanson
  • le nom du groupe
  • une description
  • l'url de l'opening
  • l'url d'une image du manga

Recherche

L'utilisateur peut effectuer une recherche simple basée sur le nom de l'anime d'un opening.
