PianoChat ~ Landing page

Présentation du concept

Ce projet reprend le concept du design d'application réalisé au cours de la formation Passerelle numérique.
Il s'agit d'une landing page de l'application de PianoChat, réseau social permettant aux joueurs de piano de communiquer entre eux dans la France entière (voir le projet PianoChat ).
Ce projet a été réalisé avec l'outil de gestion de contenu WordPress.

Objectifs

La landing page doit :

  • donner envie à l’utilisateur de télécharger l’application
  • contenir des blocs d'articles aux champs personnalisés

Couleurs

Technologies

Typographie

  • Chewy, serif : originale, utilisée pour le titre
  • Open sans, sans-serif : lisible, utilisée pour les grands et petits textes

N.B. : le projet étant hébergé sur un serveur de développement gratuit, vous pouvez voir s'afficher une erreur 504. En cas d'erreur, rafraichissez la page avec les touches CTRL Shift R.

Construction de la page avec un page builder

La landing page devait reprendre le même design que le projet de design d'application réalisé au cours de la formation passerelle numérique.

La page a été construite via l'extension Elementor qui permet d'agencer le site de manière efficace.

Mise en place d'un custom post type

Par défaut, WordPress est fourni avec deux types de contenus que sont les articles et les pages. Le custom post type permet de créer du contenu différent grâce à des champs personnalisés qui sont les custom post types.

Ici, Les champs sont des musiques jouables au piano avec toutes les informations les concernants.

Les champs crées sont :

  • le titre de la chanson
  • le compositeur
  • le niveau
  • l'année
  • la clé
  • le tempo
  • la description
