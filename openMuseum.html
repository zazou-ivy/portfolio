Open museum

Présentation du concept

Open museum est une application web permettant de visualiser les musées de la région PACA.
L'application se base sur une architecture REST et a été développée via la stack MEAN : MongoDB (Atlas), Express, Angular et Node.

Objectifs

  • l'application doit afficher tous les musées de la région PACA de la base de données publique open data
  • Les musées doivent posséder une fiche d'informations avec tous les détails du musée
  • les musées doivent posséder une carte indiquant l'adresse du musée via le module leaflet

Aperçu du site

Accueil

La page d'accueil permet à l'utilisateur de visualiser la liste des musées de la région PACA. Chaque musée contient des informations sur celui-ci comme l'adresse, le numéro de téléphone etc. Pour avoir plus de détails, l'utilisateur doit alors cliquer sur le lien "view details".

Cette page permet d'obtenir des renseignements plus précis sur le musée en question comme une carte géographique de l'emplacement du musée via le module leaflet.
